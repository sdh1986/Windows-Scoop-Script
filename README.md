# Windows-Scoop-Script

## 介绍:

此脚本用于在线安装 Scoop 及添加常用的 Bucket.

## 软件架构:

此脚本使用 Powershell 编写

## 安装教程:

使用 PowerShell5.1 或更高版本运行下面命令

```
irm https://gitee.com/sdhsparke/windows-scoop-installer/raw/master/Online-ScoopInstaller.ps1 | iex ; exit
```
